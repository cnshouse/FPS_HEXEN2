Thanks for purchase 'Gun Race' add on for MFPS 2.0!
Version 1.2.1

Require:

- MFPS 2.0 version 1.6++
- Unity 2017.4++

Get Started:

- After import the package inside a MFPS 2.0 project, go to (Toolbar) MFPS -> Addons -> GunRace -> Enable
- Now in each map scene integrate the addon, for do it simple: open the scene and click on (toolbar) MFPS -> Addons -> GunRace -> Integrate
- After this you should be able to select the game mode in the 'Host Room' menu of the lobby.

Add Weapons:

- After integrate the addon in a map you will see the GunRace game object inside the GameModes object in hierarchy, select it
  and in the inspector of bl_GunRace -> Gun Order List -> you have to add all the weapons that will be in the game mode in the order that players
  will get them.

How to play?

- in a all vs all mode, players upgrade their weapon by eliminating enemies, players win the match begin the first to reached and kill a enemy with the last weapon,
player start with a gun and get new different one after killed other players.
